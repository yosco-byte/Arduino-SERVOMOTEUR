# Arduino-SERVOMOTEUR

Ce projet permet de contrôler un servomoteur avec un bouton poussoir. Appuyez une fois pour aller à 180°, appuyez à nouveau pour revenir à 0
Il a été fait en formation.

## 🛠️ Matériel Requis
* 1 x Arduino (Uno, Nano ou Mega)
* 1 x Servomoteur 
* 1 x Bouton poussoir
* Fils de connexion
